Demo for table column group.

Notes:

  • frame property of table defines which borders will be displayed. Available values are:
    • void: no border
    • above: top
    • below: bottom
    • hsides: top & bottom
    • vsides: left & right
    • rhs: right
    • lhs: left
    • box or border: all the four edges
  • rules property of table defines which inner borders will be displayes. Available values are:
    • none: no border
    • rows: between every lines
    • cols: between every cols
    • groups: between every groups & table header/body
    • all: every lines/cols
  • nowrap property of td is not recommended which may cause width property of table take no efforts
